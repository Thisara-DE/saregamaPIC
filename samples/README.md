# Sample sheets

Drop scanned/photographed hand-written sargam sheets here (JPG/PNG/HEIC/PDF).

Used to refine the recognition requirements against real handwriting before
Phase 2 prompt design:

- verify notation standard v1 covers everything on real paper
- catalogue section delimiters (for `//`), `[ ]` usage, any Sinhala lyrics
- study the risky mark pairs: dot-below vs dash-below, dot vs dash above M
- later: private eval set for recognition accuracy

These images are personal data — they stay in this repo folder and are never
committed to any public remote.
